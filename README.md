# Projeto Meraki
Repositório TCC - Larissa Nóbrega e Nicoly Farias

O conceito do **Meraki** vem do grego que significa fazer algo com a alma, criatividade ou amor. É colocar a "essência de si mesmo" e dedicação em tudo o que se faz, deixando um pedaço de sua alma em cada detalhe, seja trabalho, arte ou tarefas cotidianas.

- **Paleta de Cores Principais:**
    - **Amarelo Claro:** Representa energia, otimismo e alegria. É a cor de destaque no logotipo, evocando a luz e o entusiasmo do lazer.
    - **Violeta Clarinho:** Traz um toque de criatividade, espiritualidade e modernidade. Esta cor é ideal para suavizar o contraste e criar uma atmosfera acolhedora.

> O Meraki é um portal dedicado a espaços de lazer, cultura e esporte, promovendo o desenvolvimento pessoal para crianças.

A equipe atua de forma colaborativa, dividindo as responsabilidades de desenvolvimento e design:

| Membro | Responsabilidades |
| :--- | :--- |
| **Larissa Nóbrega** | Desenvolvimento de funcionalidades, integração com banco de dados e documentação técnica. |
| **Nicoly Farias** | Design de interfaces, estruturação de layout e experiência do usuário. |

- **Início:** Outubro de 2025
